# Orbital Glass CTA

> A tiny playground with real production bones.

## What this is

A daily **Compotent** frontend component build created in under an hour using:

- Alpine.js interactions
- Three.js visual effect layer
- Tailwind utility styling
- Chart.js metrics block

## Features

- Action button + modal flow
- Micro interaction toggle effect
- Data chart section
- Animated 3D background accent

## Local run

```bash
python3 -m http.server 4312
# open http://localhost:4312
```

## Validation

Playwright smoke test confirms:

- page title exists
- CTA button exists
- modal opens on click
